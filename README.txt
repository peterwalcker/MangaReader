This app was created for read manga siply and comfortable.

Controls:
Next page - right arrow key or left mouse click on the right side of the screen
Previous page - left arrow key or left mouse click on the left side of the screen
Increase page size - num+ or right mouse down + mouse wheel up
Decrease page size - num- or right mouse down + mouse wheel down
Scroll page up - up arrow key or mouse wheel up
Scroll page down - down arrow key or mouse wheel down
Open file - space or mouse wheel click
Open folder (include all subfolders) - ctrl + space or right mouse down + mouse wheel click
(also you may use "open with" function)
Exit app - Escape or right mouse down + left mouse double click
